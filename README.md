# QMetryServices

This is QMetryServices project with Maven directory structure.
 
The 'config' directory contains testrun_config.xml file, and is a place holder for configuration files.

The 'resources' directory contains all required resources including properties files and data files, and is a place holder for other resources.

The 'src' directory contains all java files and is a place holder for other java files.

The 'test-results' directory contains result files.

The 'scenarios' directory is the default place holder for all the scenario files. 

To change/modify dependencies check pom.xml
To run the project, from command prompt go to project home and run mvn. Open dashboard.htm to view results.

Note: You need to download and set required driver property in application.properties file with driver binary path.

Please refer https://qmetry.github.io/qaf/
